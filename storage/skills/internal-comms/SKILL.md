# Internal Communications Templates Skill

This skill provides templates and patterns for creating internal communications documents like status reports, newsletters, FAQs, and team updates.

## Document Types

### Status Reports
Structure:
- **Executive Summary**: High-level overview (2-3 sentences)
- **Key Accomplishments**: Bulleted list of completed items
- **In Progress**: Current work items with status
- **Blockers/Risks**: Any issues or concerns
- **Next Steps**: Planned actions

### Newsletters
Structure:
- **Header**: Title, date, team/department name
- **Featured Story**: Main highlight (1-2 paragraphs)
- **Updates Section**: Multiple short updates (3-5 items)
- **Announcements**: Important notices
- **Upcoming Events**: Calendar items

### FAQs
Structure:
- **Introduction**: Brief context
- **Questions & Answers**: 
  - Clear, concise questions
  - Detailed but scannable answers
  - Use formatting (bold, lists) for readability
- **Contact Information**: Who to reach for more info

## Formatting Guidelines

1. **Use Markdown** for structure:
   - Headers (# ## ###) for sections
   - Bullet points for lists
   - Bold (**text**) for emphasis
   - Code blocks for technical details

2. **Be Concise**:
   - Keep paragraphs short (2-3 sentences)
   - Use bullet points for lists
   - Break up long content with headers

3. **Be Clear**:
   - Use plain language
   - Avoid jargon unless necessary
   - Define acronyms on first use

## Templates

### Status Report Template
```markdown
# [Project/Team] Status Report
**Date**: [Date]
**Period**: [Week/Month/Quarter]

## Executive Summary
[Brief overview of status]

## Key Accomplishments
- [Accomplishment 1]
- [Accomplishment 2]

## In Progress
- [Item 1] - [Status]
- [Item 2] - [Status]

## Blockers/Risks
- [Blocker/Risk 1]
- [Blocker/Risk 2]

## Next Steps
- [Action 1]
- [Action 2]
```

### Newsletter Template
```markdown
# [Team] Newsletter - [Date]

## Featured Story
[Main highlight with 1-2 paragraphs]

## Updates
### [Update Title]
[Brief description]

### [Update Title]
[Brief description]

## Announcements
- [Announcement 1]
- [Announcement 2]

## Upcoming Events
- [Event 1] - [Date]
- [Event 2] - [Date]
```
